# 付费自习室管理系统 java44

## 项目概述

本项目是基于Springboot和Vue的前后端分离的付费自习室管理系统，旨在为用户提供便捷的自习室预约与管理功能。

## 技术栈

- 后端：Java、Springboot、MyBatis、MySQL
- 前端：Vue、jQuery、HTML

## 功能模块

### 后台功能

- 首页信息统计
- 用户管理
- 自习室信息管理
  - 自习室信息
  - 自习室类型
- 预约订单管理
- 取消订单管理
- 评价信息管理
- 轮播图管理

### 前台功能

- 首页信息展示
- 公告信息
- 自习室信息
- 个人中心

## 系统角色

- 管理员：负责后台管理操作
- 用户：使用前台功能进行自习室预约等操作

## 运行环境

- 后端：Java 1.8+、MySQL 5.7+
- 前端：现代浏览器

## 部署步骤

（此处假设用户已了解基本的部署步骤，具体部署细节请参考相关文档）

1. 准备后端运行环境（Java、MySQL等）
2. 部署后端项目（部署Springboot应用）
3. 准备前端运行环境（Vue项目编译）
4. 部署前端项目（将编译后的Vue文件部署到Web服务器）
